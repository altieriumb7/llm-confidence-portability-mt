# Confidence–Difficulty Mismatch in MT Evaluation (Code Repository)

This repository is the **code + artifact-generation** workspace for the project.

## Authoritative bundled raw snapshot

For deterministic offline regeneration, the authoritative raw inputs are:

- `runs/snapshots/20260228_000439/raw/*.jsonl`

Use these files as the source of truth when reproducing aggregate outputs without live API calls.

## Reproduction entrypoints

### 1) Offline deterministic regeneration (recommended)

Regenerate all manuscript-facing derived artifacts from bundled snapshot data:

```bash
bash scripts/generate_all_artifacts.sh
```

This runs, in order:

1. `scripts/regenerate_tables.sh` (stages snapshot raw JSONL, regenerates aggregate CSV/JSON outputs)
2. `scripts/regenerate_figures.sh` (regenerates plots + mismatch examples markdown)

### 2) Narrower regeneration commands

Regenerate non-plot aggregate artifacts only:

```bash
bash scripts/regenerate_tables.sh
```

Regenerate figures and paper examples from an existing aggregated dataframe:

```bash
bash scripts/regenerate_figures.sh
```

Legacy one-shot script (also snapshot -> aggregates -> figures):

```bash
bash scripts/generate_figures.sh
```

## Which steps require live APIs

Pipeline stage boundaries are implemented in `run_repro.sh`.

- **Requires live APIs + provider keys:**
  - Step 2: `src/02_translate_and_confidence.py`
  - Triggered by `bash run_repro.sh --mode step2` (or `--mode all`)
  - Required env vars:
    - `OPENAI_API_KEY`
    - `ANTHROPIC_API_KEY`
    - `GEMINI_API_KEY` (or `GOOGLE_API_KEY`)

- **Offline-compatible (no live API calls):**
  - `src/03_features_and_metrics.py`
  - `src/04_analysis_and_plots.py`
  - Downstream analyses (`src/05_*`, `src/06_*`, `src/07_*`, `src/08_*`) when run from existing local artifacts.

## Exact commands (common workflows)

Install dependencies:

```bash
python3 -m pip install --root-user-action=ignore -U pip
python3 -m pip install --root-user-action=ignore -r requirements.txt
```

Offline regenerate from snapshot (tables/aggregates only):

```bash
bash scripts/regenerate_tables.sh
```

Offline regenerate figures from regenerated aggregates:

```bash
bash scripts/regenerate_figures.sh
```

Full pipeline wrapper (may include live APIs depending on mode):

```bash
bash run_repro.sh --help
bash run_repro.sh --mode all
```

## Manuscript-adjacent outputs generated by shell scripts

These shell scripts generate binary artifacts and other derived outputs:

- `scripts/regenerate_tables.sh`
  - writes `runs/aggregated/dataframe.csv`
  - writes `runs/aggregated/results_by_model.json`
  - writes `runs/aggregated/summary_table.csv`
  - writes `runs/aggregated/meta.json`
- `scripts/regenerate_figures.sh`
  - writes `figures/*.png` and `figures/*.pdf`
  - writes `paper/top_mismatch_examples.md`
- `scripts/generate_all_artifacts.sh`
  - orchestration wrapper for both scripts above
