# Reproducibility Checklist (Final)

This checklist defines minimal and extended reproducibility paths, with explicit separation between offline-regenerable stages and API-required stages.

## A) Environment requirements

- OS: Linux/macOS shell environment
- Python: `python3` on PATH
- Dependencies:

```bash
python3 -m pip install --root-user-action=ignore -U pip
python3 -m pip install --root-user-action=ignore -r requirements.txt
```

- Optional for full manuscript compile: LaTeX toolchain (`latexmk`, `pdflatex`, `bibtex`)
- Optional for Step 1 dataset acquisition: `sacrebleu` testset availability

## B) Seeds/config/raw data locations

- Primary config: `configs/models.yaml`
- Dataset/sample seed for reported run context: seed `123` (recorded in manuscript/meta workflow)
- Authoritative bundled raw snapshot:
  - `runs/snapshots/20260228_000439/raw/*.jsonl`

## C) Offline reproducible path (no live APIs)

### C1. One-command offline regeneration (recommended)

```bash
bash scripts/generate_all_artifacts.sh
```

This runs:

1. `bash scripts/regenerate_tables.sh`
2. `bash scripts/regenerate_figures.sh`

### C2. Equivalent explicit two-step path

```bash
bash scripts/regenerate_tables.sh
bash scripts/regenerate_figures.sh
```

### C3. Exact non-text artifact regeneration command scope

Binary and non-text artifacts are regenerated by:

- `bash scripts/regenerate_figures.sh`
  - regenerates `figures/*.png` and `figures/*.pdf`

(Use after aggregates exist; `generate_all_artifacts.sh` ensures ordering.)

### C4. Quick checks

```bash
python3 - <<'PY'
import csv, json
from pathlib import Path
meta=json.loads(Path('runs/aggregated/meta.json').read_text())
rows=list(csv.DictReader(open('runs/aggregated/dataframe.csv', encoding='utf-8')))
summary=list(csv.DictReader(open('runs/aggregated/summary_table.csv', encoding='utf-8')))
results=json.loads(Path('runs/aggregated/results_by_model.json').read_text())
print('dataframe_rows:', len(rows))
print('meta_n:', meta.get('n'))
print('summary_models:', len(summary))
print('results_models:', len(results))
PY
```

## E) Manuscript compile workflow after artifacts are ready

Manuscript entrypoint:

- `revised_submission_with_new_results.tex`

Compile:

```bash
latexmk -pdf -interaction=nonstopmode revised_submission_with_new_results.tex
```

Fallback compile sequence:

```bash
pdflatex revised_submission_with_new_results.tex
bibtex revised_submission_with_new_results
pdflatex revised_submission_with_new_results.tex
pdflatex revised_submission_with_new_results.tex
```

## F) Limitations / unsupported byte-level reproducibility

- API-backed raw generation may vary over time (provider/model drift).
- Binary figures are script-regenerated and semantically reproducible, but may not be byte-identical across platforms/toolchains.
- Optional COMET-dependent analyses require local availability of relevant models/runtime.
