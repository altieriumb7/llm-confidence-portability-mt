# Paper build and paper-facing assets

## Manuscript entrypoint (repository root)

Compile from repository root:

```bash
latexmk -pdf -interaction=nonstopmode revised_submission_with_new_results.tex
```

## Mandatory pre-build generation

Regenerate all paper-facing assets first:

```bash
bash scripts/generate_paper_assets.sh
```

This command regenerates and synchronizes:

- figures (`figures/*`)
- paper examples markdown (`paper/top_mismatch_examples.md`)
- manuscript tables (`tables/*.tex`)
- supporting aggregated outputs (`runs/aggregated/*`)

and runs `tools/consistency_check.py`, which hard-fails on table drift.

## Source mapping for manuscript tables

- `tables/summary.tex` ← `runs/aggregated/summary_table.csv`
- `tables/corr.tex` ← `runs/aggregated/results_by_model.json` (correlations)
- `tables/robustness.tex` ← `runs/aggregated/results_by_model.json` (mismatch@0.9 variants)
- `tables/calibration.tex` ← `runs/aggregated/calibration/calibration_summary.json`
- `tables/metric_robustness.tex` ← `runs/aggregated/metric_robustness/metric_robustness_summary.json`

All five are generated by `tools/export_latex_tables.py`.

## Bibliography status

The manuscript includes:

```tex
\bibliography{references,added_refs}
```

Only `added_refs.bib` is present in this repository. `references.bib` is missing from the bundled artifact. Full bib resolution therefore requires adding that external file.

See `paper/TODO_missing_bibliography.md` for the explicit blocker note.
