# Reviewer Artifact Guide

This guide is the reviewer-facing entrypoint for reproducing the paper artifact.

## 1) Scope at a glance

### Fully reproducible offline (from bundled snapshot)
- Regenerate aggregated analysis artifacts from bundled raw JSONL snapshot
- Regenerate manuscript-facing tables (`tables/*.tex`)
- Regenerate figures (`figures/*.png`, `figures/*.pdf`)
- Regenerate supplementary analyses (calibration, selective analysis, parse audit, metric robustness)
- Regenerate semantic-audit exports for high-confidence mismatch review
- Regenerate external comparator baseline outputs (fully offline)
- Regenerate prompt-sensitivity status manifests (offline status + optional live paths)
- Run consistency checks across manuscript/assets

### Requires live API credentials
- Fresh translation/confidence collection (`src/02_translate_and_confidence.py`)
- Full end-to-end rerun from scratch against provider APIs

## 2) Recommended command paths

### Canonical offline reproduction (recommended)
```bash
bash scripts/reproduce_offline_artifact.sh --skip-manuscript
```

### Offline reproduction + manuscript build attempt
```bash
bash scripts/reproduce_offline_artifact.sh
```

### Optional live/API rerun path
```bash
bash run_repro.sh --mode all
```

> Live/API reruns are **not** guaranteed to reproduce identical model outputs due to external provider changes and API-side non-stationarity.

## 4) Environment

### Python deps
```bash
python3 -m pip install -r requirements.txt
```

### Containerized path (optional)
A minimal reproducible container is provided:
```bash
docker build -t mt-confidence-artifact .
docker run --rm -it mt-confidence-artifact bash scripts/reproduce_offline_artifact.sh --skip-install --skip-manuscript
```

## 5) Validation
Run all artifact consistency checks:
```bash
bash scripts/validate_artifact.sh
```

This runs:
- `tools/test_parse.py`
- `tools/export_latex_tables.py --check`
- `tools/consistency_check.py`

Current checks verify:
- manuscript table drift against regenerated LaTeX tables,
- manuscript `\input{tables/...}` and figure-file wiring,
- key scalar alignment (`seed`, `n`, `tau`) between manuscript text and `runs/aggregated/meta.json`,
- numeric consistency across `summary_table.csv`, `results_by_model.json`, calibration/robustness tables, and selective-threshold exports,
- metadata integrity (`config_hash_sha256` vs active config; provenance field structure).

Current checks do **not** verify:
- bibliography completeness/resolution in TeX toolchains,
- narrative correctness of manuscript prose,
- semantic correctness of examples/figures.

## 6) Known limitations
- The bundled snapshot supports deterministic offline regeneration of analysis artifacts only.
- COMET/xCOMET checkpoints are not bundled in this artifact; enabling learned-metric scoring requires external downloads.
- A deterministic semantic-audit scaffold is bundled and fully reproducible offline; completed human labels are optional external inputs under `runs/annotations/semantic_audit/*.csv`.
- Prompt variants beyond `canonical_v1` are configured and executable, but data for those variants requires optional live reruns with API credentials.
- Live API reruns are not exact reproductions of the snapshot.
- `references.bib` is included as an empty compatibility file; canonical bibliography metadata is not fully bundled in this zip.
